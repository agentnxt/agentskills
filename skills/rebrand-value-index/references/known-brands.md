# Known High-Recognition Brands

Apps on this list automatically score −3 for brand recognition in the RVI.

A brand qualifies if it is: widely known in the self-hosting community,
prominently featured on awesome-selfhosted, r/selfhosted, or similar, OR
has an active foundation/vendor that enforces the name.

## How to use this list

Match the upstream repo name or org name (case-insensitive) against entries here.
Partial matches count — e.g. `rocket.chat` matches `rocketchat`.

If the app is clearly prominent in self-hosting circles but not listed,
apply −3 anyway and add it to this list for future runs.

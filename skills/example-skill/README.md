# Example Skill

Example Skill is a minimal reference package for the AGenNext Skill Registry.

Use this folder as a starting point when creating a new registry entry. Replace the example content with real instructions, setup steps, runtime requirements, permissions, dependencies, and usage examples for your skill.

## What it does

This example demonstrates the expected registry structure:

- `SKILL.md` for assistant-facing instructions
- `skill.json` for machine-readable metadata
- optional `README.md` for human-facing setup and usage docs
- optional `CHANGELOG.md` for release notes

## Requirements

This example has no runtime requirements.

Real skills should document:

- required Claude or Claude Code environment
- APIs, MCP servers, databases, or tools
- required environment variables
- expected input and output formats

## Install

Copy this skill folder into the target skill manager or registry consumer.

## Permissions

This example requires no network, filesystem, shell, browser, or secret access.

## Limitations

This is only a template package and should not be treated as a production skill.

# Example Skill

Use this skill as a reference template for new entries in the AGenNext Skill Registry.

## Purpose

This skill demonstrates the minimum shape of a registry skill package. It is not intended for production use.

## When to use

Use this example when creating a new skill folder and manifest.

## Instructions

When adapting this template:

1. Replace this file with clear assistant-facing instructions.
2. Describe when the skill should and should not be used.
3. Document required inputs and expected outputs.
4. Document any tools, files, APIs, databases, or credentials the skill requires.
5. Keep operational steps concrete and testable.

## Output

Return concise Markdown unless the user requests another format.

📥 Manual Installation

To install a skill manually in Claude:

# Download the skill file
curl -L -o /path/to/skill.skill https://raw.githubusercontent.com/agentnxt/agentskills/main/dist/[skill-name].skill

# Or install via Claude's skill manager
/load-skill https://raw.githubusercontent.com/agentnxt/agentskills/main/dist/[skill-name].skill

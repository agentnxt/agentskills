## Microsoft Lifecycle Policy

Two main policies govern support timelines:

### Fixed Lifecycle Policy (on-premises products)
- **Mainstream Support** — full support, new features accepted
- **Extended Support** — security updates only (free), paid non-security updates
- No new design changes or features during Extended Support
- Ref: support.microsoft.com/en-us/hub/4095338/microsoft-lifecycle-policy

### Modern Lifecycle Policy (cloud + Microsoft 365)
- Continuous updates; customers must stay current
- No fixed end date — governed by staying on supported versions
- Microsoft 365 Apps governed by this policy

Products ending/transitioning support in **2026**:
- Several products moving from Mainstream → Extended Support
- Publisher retiring in 2026
- Monthly Enterprise Channel M365 final feature update: October 13, 2026
- Check full list: support.microsoft.com/en-us/help/4538624/products-reaching-end-of-support-for-2026

## Support Channel Types

### 1. Self-Service (Free)
- **support.microsoft.com** — article search, step-by-step guides, troubleshooters
- **Microsoft Q&A** (answers.microsoft.com) — community technical Q&A, Microsoft-moderated
- **Microsoft Answers** — consumer product forums
- **Microsoft Tech Community** (techcommunity.microsoft.com) — IT Pro + developer forums

### 2. Microsoft Q&A (answers.microsoft.com)

- .NET | Azure | Entra | M365 & Office | Microsoft Teams | Outlook | SharePoint
- SQL Server | Surface | Visual Studio | Windows (Business) | Windows (Home)
- Microsoft Edge | Microsoft Security | Developer Technologies | Microsoft Advertising

Sub-communities:
- Microsoft Learn community → learn.microsoft.com/en-us/training/learn-community
- Dynamics community → community.dynamics.com
- Learning rooms → aka.ms/learningrooms

### 3. Microsoft 365 Admin Support
- M365 Admin Portal support: portal.office.com/AdminPortal#/support
- Admins can open tickets, chat with support, and get call-back support directly from admin center

### 4. Azure Support Plans

| Plan | Who It's For | Billing/Break-fix | Advisory | Response (Sev A) |
|---|---|---|---|---|
| **Basic** (free) | All Azure customers | ✅ | ❌ | No SLA |
| **Developer** | Trialing / non-production | ✅ | Limited | 8 business hours |
| **Standard** | Production workloads | ✅ | ❌ | 2 hours |
| **Professional Direct** | Business-critical | ✅ | ✅ | 1 hour |
| **Microsoft Unified** | Enterprise | ✅ | ✅ | 15 min (Sev 1 critical) |

Azure support portal: portal.azure.com → Help + Support
Plans comparison: azure.microsoft.com/en-us/support/plans

### 5. Microsoft Unified Enterprise Support
The flagship enterprise support offering (replaces old Premier Support).

Key features:
- **15-minute** initial response for Azure Critical Business System Down (Severity 1)
- 24x7 coverage for Severity A and B
- Coverage for entire Microsoft technology footprint (Azure, M365, Dynamics, Power Platform, Windows)
- Dedicated Customer Success Account Manager (CSAM)
- As-needed advisory consults
- Proactive services (assessments, workshops, risk mitigation)
- Learning Manager role in Services Hub for team upskilling

Pricing: Rate applied to historical annual IT spend (Product Spend = P)
- Cloud services (M365, Dynamics 365) + Azure consumption
- Graduated pricing starting at 8–10%

⚠️ **Important 2026 change**: Microsoft is transitioning **Services Hub → Microsoft Engage Center** (February–April 2026). Customers must prepare at aka.ms/sh-getready.

Severity levels for Unified:
| Severity | Definition | Response |
|---|---|---|
| A (Critical) | Complete loss of core business function | 15 min (Azure Sev 1) / 2 hours (other) |
| B (High) | Significant degradation, workaround exists | 4 hours |
| C (Moderate) | Some impact, workaround available | 8 business hours |

Languages for 24x7 support: English, Spanish, French, German, Italian, Portuguese, Traditional Chinese, Simplified Chinese, Japanese.

Unified Enterprise details: microsoft.com/en-us/microsoft-unified/plan-details
USSD (Services Description, Nov 2025): aka.ms/UnifiedServicesDescription

## Support Escalation Paths

```
Issue arises
      │
      ├── Self-service docs (support.microsoft.com) → 80% of issues resolved here
      │
      ├── Microsoft Q&A / Tech Community → community answers
      │
      ├── M365 Admin Center (in-product support) → for admins/IT pros
      │
      ├── Azure Portal Help + Support → for Azure issues
      │
      ├── Professional Direct → 1-hour response, advisory included
      │
      └── Microsoft Unified Enterprise → CSAM, 15-min Sev 1, proactive services
```

## MSFT-X Support Routing Guide

When a user asks for help with a Microsoft product:

| Situation | MSFT-X Recommendation |
|---|---|
| End user issue (Office, Teams, OneDrive) | → support.microsoft.com/[product] |
| IT admin issue (M365 tenant, Intune, Azure) | → Admin Center in-product support |
| Developer / API / code issue | → Microsoft Q&A (answers.microsoft.com) |
| Azure outage / service health | → status.azure.com |
| M365 service health | → admin.microsoft.com → Health → Service Health |
| Enterprise escalation | → CSAM via Services Hub / Engage Center |
| Billing dispute | → support.microsoft.com/microsoft-store-and-billing |
| Reporting scam | → microsoft.com/reportascam |
| Windows 10 still in use | → ⚠️ EOL Oct 2025, upgrade to Windows 11 immediately |
| Office 2016/2019 in use | → ⚠️ EOL Oct 2025, migrate to Microsoft 365 |
| Publisher in use | → ⚠️ Retiring 2026, plan migration now |

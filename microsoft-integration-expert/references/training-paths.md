## Certification Ladder

```
Fundamentals (900-level) ─── entry, no prerequisites
         ↓
Associate (100–400 level) ─── role-based, hands-on
         ↓
Expert (400–600 level) ─── deep expertise
         ↓
Specialty ─── niche domains (SAP, IoT, Cosmos DB)
         ↓
Applied Skills ─── scenario-based, no exam, faster
```

All Microsoft certs renew annually via FREE online renewal assessment (no re-exam).
Certification portal: https://learn.microsoft.com/en-us/credentials/
Catalog API: https://learn.microsoft.com/api/catalog/

## MSFT-X Routing Guide for Training Questions

| User Need | MSFT-X Response |
|---|---|
| New to Microsoft/cloud | AZ-900 free self-paced modules + Virtual Training Days |
| Wants specific role | Match career path → Associate cert study path |

| Getting certified | Cert study path + practice assessment + ILT option |
| Prove a skill quickly | Applied Skills (hands-on, no exam) |
| Senior architect | Expert certs (AZ-305, SC-100, PL-600) or Specialty |

| Org-wide AI skilling | AI Learning Hub → learn.microsoft.com/en-us/ai/ |

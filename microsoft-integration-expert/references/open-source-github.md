## OSCI — Open Source Contributor Index (opensourceindex.io)

**What OSCI measures:**
- Tracks GitHub commits by corporate employees (via email domain matching)
- **Active Contributors:** employees with 10+ commits in the current year
- **Total Community:** employees with at least 1 commit
- Covers ~320 commercial organizations across 12 industries
- Built by EPAM Systems, hosted at opensourceindex.io, powered by Azure (Data Factory, Functions, Databricks)

**Microsoft's OSCI ranking:**
- **#1 consistently from 2016–2021** in Active Contributors
- **Neck-and-neck with Google** at the top since 2021
- Historical peak: ~4,400+ Active Contributors (employees with 10+ commits/month)
- Microsoft + Google + Red Hat = consistent top 3 since 2016

**OSCI methodology note:** Tracks by email domain — *microsoft.com* commits are matched. Microsoft employees contributing to open source under personal emails are NOT counted, meaning actual contribution volume is likely higher.

**Competitor comparison (historical):**
